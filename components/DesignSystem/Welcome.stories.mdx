# Design System

Welcome to the Design System component library! This collection provides a comprehensive set of reusable UI components built on top of Mantine.

## Philosophy

Our design system follows these key principles:

- **T-shirt Sizing**: Components use intuitive size names (xs, sm, md, lg, xl) instead of numeric values
- **Consistency**: All components follow the same patterns and conventions
- **Flexibility**: While providing sensible defaults, components allow customization when needed
- **Accessibility**: Built with accessibility in mind using Mantine's foundation

## Component Categories

### Typography
- **Text**: Text component with t-shirt sizing
- **Title**: Heading component with both t-shirt sizes and semantic heading levels

### Data Display
- **ThemeIcon**: Icon container with custom xxl size (58x58px) and color variants

### Inputs
- Various form input components with consistent sizing

### Buttons
- Button components with consistent styling and sizing

### Navigation
- Navigation components for consistent user flows

### Overlays
- Modal, tooltip, and other overlay components

### And More...
- Additional components across multiple categories
